# AI Interview Preparation System

A simple AI-powered interview preparation system built using Python and Streamlit.

## Features

- Practice common interview questions
- Enter answers through a user-friendly interface
- Automatic scoring based on response length
- Simple and interactive design

## Technologies Used

- Python
- Streamlit

## Project Files

- app.py
- questions.py
- score.py
- requirements.txt

## How to Run

1. Install Streamlit

pip install streamlit

2. Run the application

streamlit run app.py
